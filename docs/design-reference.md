# BioMap — референс дизайна (описание скриншотов от пользователя)

## Общие ощущения
- Очень светлый зелёно-белый фон, белые карточки, скруглённые углы, много воздуха
- Primary цвет — спокойный зелёный (кнопки, прогресс, активные состояния)
- Текст почти чёрный, минимум цветовых акцентов
- Иконки простые, линейные, в мягких пастельных кружках (жёлтый/синий/зелёный/красный)
- Шрифт — простой sans-serif, чёткая иерархия размеров

## Экраны из референса
1. **Главная** — логотип BioMap + колокольчик уведомлений, приветствие по имени, большая карточка "Что изучаем сегодня?" с прогрессом урока и зелёной кнопкой "Продолжить", 3 маленькие карточки (Мои ошибки / Повторение / Прогресс), блок "Быстрый доступ" (Темы / По классам / Тесты (Milliy) / Справочник), bottom nav из 5 пунктов.
2. **Сквозные темы** — табы (Все темы / Изучаю / Закрепил), список тем с иконкой, диапазоном классов и % прогресса + полоска прогресса.
3. **Тема → классы** — горизонтальные табы классов (5–11), ниже вертикальный таймлайн связи темы с каждым классом (кружки разных цветов = разные подтемы по классам), кнопка "Изучить тему".
4. **Микроурок** — верхние табы (Теория / Схемы / Вопросы), крупная подписанная SVG-схема (напр. клетка в разрезе с подписями органелл), текстовый блок объяснения, навигация "Назад / 3 из 8 / Далее".
5. **Тест** — прогресс-бар "Вопрос 4 из 10", варианты ответов A/B/C/D, выбранный правильный вариант подсвечен зелёным с галочкой, кнопки "Пропустить" / "Ответить".
6. **Подсказка** — "Подсказка 1 из 3", текст наводящего вопроса, статистика "Ошибался в похожих вопросах: 3 раза", кнопка "Следующая подсказка".
7. **Мои ошибки** — табы (Темы / Вопросы), список тем с процентом ошибок (напр. "Ошибок: 8 из 15 — 53%"), кнопка "Смотреть все".
8. **Повторение** — расписание на день по времени (10:00, 12:00, 16:00, 20:00) с темой и кнопкой "Повторить", блок "Серия: 5 дней 🔥".

## Вывод для Tailwind-темы
- background: очень светлый зелёно-белый (напр. #F3FBF6 / эквивалент)
- card: белый (#FFFFFF), border очень светлый серый
- primary: спокойный зелёный (напр. #16A34A / около этого тона)
- secondary: мягкий зелёный (светлее primary, для фона плашек/бейджей)
- text: почти чёрный (#1A1A1A), без чистого #000
- status-colors: отдельные мягкие цвета только для ошибка/успех/предупреждение
